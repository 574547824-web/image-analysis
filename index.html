工作流程

1

接收图片

获取用户上传的图片（URL 或本地路径）

2

解析图片内容及构图

  • 主体对象：人物、动物、物体等画面核心元素
  • 场景环境：室内/室外、地点类型、空间特征
  • 动作状态：正在进行的动作、情感表达、人物互动
  • 关键细节：表情、服饰、道具、背景元素
  • 景别：远景、全景、中景、近景、特写
  • 视角：平视、仰视、俯视、侧视、鸟瞰、低角度
  • 构图方式：三分法、对称构图、引导线、框架构图等
  • 画面平衡：视觉重心、空间分布、元素排布
3

识别图片画风

  • 风格类型：写实、超写实、漫画、插画、油画等
  • 画派特征：印象派、超现实主义、立体主义等
  • 技术特点：笔触风格、线条特征、光影处理
  • 时代特征：古典、现代、当代、复古、未来主义
  • 氛围调性：明亮、阴暗、温暖、冷峻、梦幻
4

提取色彩参数

  • 一级调色：阴影、中间调、高光、全局
  • 对比与曝光：对比度、曝光、Gamma、增益、偏移
  • 色轮偏移：阴影/中间调/高光色相与饱和度偏移
  • 曲线：RGB曲线、红通道、绿通道、蓝通道
  • 胶片模拟：胶片类型、颗粒感、暗角、光晕
  • 风格化工具：锐化、模糊、去噪、纹理

色彩参数

一级调色

【阴影】色相 0-360°，饱和度 0-100%，亮度 0-100% 【中间调】色相 0-360°，饱和度 0-100%，亮度 0-100% 【高光】色相 0-360°，饱和度 0-100%，亮度 0-100%

对比与曝光

对比度：0-100% 曝光：-5.0 至 +5.0 EV Gamma：0.1 至 3.0 增益：0-100% 偏移：-100 至 +100

胶片模拟

胶片类型：柯达 Portra、富士 Velvia 颗粒感：0-100% 暗角：0-100% 光晕：0-100%

风格化工具

锐化：0-100% 模糊：0-100% 去噪：0-100% 纹理：0-100%

使用方法

# 解析本地图片 python scripts/analyze_image.py --image /path/to/image.jpg # 解析网络图片 python scripts/analyze_image.py --image https://example.com/image.jpg # 指定输出目录 python scripts/analyze_image.py --image /path/to/image.jpg --output-dir ./output # 生成简化版 prompt python scripts/analyze_image.py --image /path/to/image.jpg --mode short

参数说明

参数 说明 默认值
--image 图片路径或 URL 必填
--output-dir 输出目录 当前目录
--mode 输出模式：full（完整）/ short（简化） full

注意事项

⚠️ 色彩参数为估算值

达芬奇调色参数是基于视觉分析的估算值，精确值需在达芬奇软件中调整
